# Sub-GHz Sensor Mesh Study (Experiment Phase 1) 

## Project Overview 
Данный репозиторий содержит материалы учебного исследования, посвященного изучению устойчивости распределенных сенсорных сетей в диапазоне 400-550 МГц. Основное внимание уделяется анализу распространения сигнала в условиях сложного рельефа и высокого уровня электромагнитного шума. 

## Research Goals 
* Исследование замираний (fading) сигнала при использовании узкополосных протоколов.
* Анализ временных характеристик (latency) при передаче коротких пакетов данных.
* Оценка эффективности псевдослучайных алгоритмов перестройки частоты (FHSS) для минимизации потерь.

## Hardware Stack 
* MCU: ESP32-C3 (RISC-V)
* RF: Semtech LR1121 (Sub-GHz / S-Band)
* Protocol: Custom implementation based on LRP concepts.

## Project Structure 
* `/src` — исходный код исследовательских нод.
* `/include` — конфигурационные файлы (параметры синхронизации).
* `/docs` — теоретические выкладки по расчету энергетического бюджета линка.

## Disclaimer 
Проект носит исключительно образовательный характер. Все эксперименты проводятся в рамках разрешенных мощностей и частотных диапазонов.
